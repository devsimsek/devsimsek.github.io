Lets Learn C++ | #1 Basics

Posted on Per 29 Nisan 2021 in Learn

Welcome to my c++ dev journey! Today i will tell you about my thoughts. So lets begin!

Why I Chose c++?

c++ is great language but its not for beginners. If you want to learn c++ as your first language please go and learn some other language. Such as python! c++ is hard and complex language but that makes c++ even more interesting for me. I really like to challenge myself in computer science thats why i chose c++.

Preparing Environment

Firstly, you need an ide. I chose CLion from jetbrains. You can still use visual studio code. Then you will need an compiler to create your applications. I use cmake which comes with CLion. If you are using apple mac osx you can use gcc compiler which comes by default.

Hello World!

C++ has a basic structure for basic terminal applications. I will give sample code down bellow this line.

#include <iostream> // including input/output stream to print console.
int main() {
    std::cout << "Hello World!" << std::endl;
    return 0;
}

Lets explain what we did here;

#include <iostream>: By adding #include c++ will include your header files. If header file names has <> these flags it means it is global header. If you want to include your own local one use "" tags to do it.

int main() {}: This is the main function that most of the compilers use. Basic structure of functions is type name(variables) {code}

std::cout << "Hello World!" << std::endl; lets look line by line; std::: std:: is a namespace for iostream. iostream is a basic input and output stream header for c++. it comes by global. std::cout: cout is a simple way to print text through console. <<: These tags are injection tags. In c++ and c we dont use +, . like most of the languages out there. << std::endl;: endl means endline. it will create new line when added into cout function.
